自主推理决策

基于 ReAct 循环，Agent 能理解你的需求，自主决定调用哪些工具、分析哪些数据，而不是死板的问答流程。

📊

位次法精准预测

利用一分一段表做位次映射，比单纯对比分数更科学。历年数据交叉验证，概率评估更可靠。

🎯

冲稳保全方案

自动生成三档梯度志愿方案，精确到每个专业的录取概率和建议顺序，告别盲目填报。

💬

多轮对话记忆

支持连续追问和条件调整。"换个城市呢？""我不想学医怎么办？"——Agent 记住上下文，灵活应变。

📜

政策智能解读

内置平行志愿、新高考选科、退档风险等政策知识库，让考生和家长不再迷糊。

🔌

数据实时可扩展

对接掌上高考 API，支持爬取最新录取数据。数据每年更新，方案永不过时。

快速开始

3 分钟跑起来，开始和 AI 志愿师对话

  Terminal 
# 克隆项目
git clone https://github.com/diyiwuyan/gaokao-agent.git
cd gaokao-agent

# 安装依赖
python -m venv .venv && source .venv/bin/activate
pip install -e .

# 配置 API Key
cp .env.example .env
# 编辑 .env 填入 LLM API Key (DeepSeek/智谱/Moonshot等)

# 初始化数据 & 启动
python main.py init-db
python scripts/seed_demo_data.py  # 灌入演示数据
python main.py ui                 # 🚀 启动 Web 界面
